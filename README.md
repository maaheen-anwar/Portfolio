# Personal Portfolio Website

A modern, dark-themed personal portfolio website built to showcase education, skills, projects, and achievements in a clean and visually engaging way.

## Tech Stack
- HTML5  
- CSS3  
- JavaScript  

## Features
- Dynamic landing page with animated role switching  
- Smooth scroll-based navigation  
- Responsive layout with modern UI styling  
- Project showcase with horizontal sliding cards  
- Clean contact section with direct links  

## Purpose
Designed to represent professional identity and highlight technical expertise through a minimal yet dynamic web presence.
